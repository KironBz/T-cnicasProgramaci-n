# SistemaGimnasio_P2_TP
Un gimnasio desea un sistema que permita registrar a sus usuarios, crear rutinas de entrenamiento personalizadas (por ejemplo: bajar de peso, ganar masa muscular, mejorar la condición física), asignar entrenadores a cada usuario y buscar usuarios, rutinas y entrenadores.

Sistema de gestión de entrenamientos - Gimnasio
Este proyecto implementa un sistema para gestionar, usuarios, entrenadores y rutinas.
